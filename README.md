
Setup instructions

```bash
git clone https://github.com/bqqbarbhg/clock-in
cd clock-in
```

Setup the server

```bash
cd clock-in/server
npm install
npm start
# Serves the API at http://localhost:8080
```

Setup the client development server

```bash
cd clock-in/client
npm install
npm run start:dev
# Serves the client at http://localhost:8000
```
